CSS Animation

Kevin Hoyt

I had a sociology professor in university who was fond of saying "People are attracted to bright, shiny things that move." Since I have yet to find evidence to the contrary, I often wonder why the web is so static. In this session you will learn about CSS3 Transitions and Animations, and be empowered to bring life to the web. Just remember that "With great power, comes great responsibility." (Ben Parker, not my university professor)

13:00 - 13:45

The Future of HTML5 Motion Design

Terry Ryan

HTML5 and CSS3 are hot, driven by an explosion of new, Internet connected devices. While they offer many new features that should allow you to do the types of things that you previously did in Flash, actually making it happen is really hard. Until now. This session will explain how Adobe Edge Preview 6 can be used to save time while creating beautiful motion and interactive content. Adobe Edge is a new tool that natively uses HTML5, CSS3 and JavaScript for fidelity across modern browsers, while enabling integration into real-world workflows.

This session will also delve into Adobe's contributions to Webkit for making more expressive web creations in the future.

15:45 - 16:30

Designing HTML for Devices

Terry Ryan

Responsive Web Design is easy, just use media queries and make everything bigger, right? Um, actually, that is a lot of it. But the rest of it is making decisions about what to shrink, what to move, what screen size do you set your queries too? All these and more are the hard stuff of mobile design. This session will explore these issues and hopefully help you answer the questions for yourself.

16:30 - 17:15

Adobe Shadow

Mihai Corlan

Everybody talks about how mobile is so important, mobile first, responsive design, and so forth. Adobe Shadow is a new tool that streamlines mobile development. It makes testing your website so easy that you will feel like you didn¹t even leave your desktop in the first place. In this session you will see how you can leverage Adobe Shadow for testing and debugging your mobile websites and web apps on Android and iOS devices.

17:30 - 18:15

PhoneGap and PhoneGap Build

Mihai Corlan

PhoneGap is an open source platform for developing mobile applications that run across different devices and operating systems. If this is not cool enough then let me add one more thing: you code using JavaScript, HTML, and CSS. In this session you will find out what PhoneGap is and how you can use it to build mobile applications. You will see PhoneGap Build in action - the easiest way to build/package your files into the native installers.
